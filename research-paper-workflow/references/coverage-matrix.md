# Coverage Matrix

This matrix defines the standard minimum coverage for "most academic research content".

| Capability | Empirical | Systematic Review | Methods | Theory |
|---|---|---|---|---|
| Framing and contribution | Yes | Yes | Yes | Yes |
| Literature and positioning | Yes | Yes | Yes | Yes |
| Study design and instruments | Yes | Optional | Yes | Optional |
| Ethics and data governance | Yes | Optional | Optional | Optional |
| Evidence synthesis / meta-analysis | Optional | Yes | Optional | Optional |
| Manuscript drafting | Yes | Yes | Yes | Yes |
| Compliance checks | Yes | Yes | Yes | Yes |
| Submission package | Yes | Yes | Yes | Yes |
| Rebuttal package | Yes | Yes | Yes | Yes |
| Code/repro support | Optional | Optional | Yes | Optional |

## Out of Scope by Default

- Field-specific legal requirements
- Institution-specific IRB portal submission
- Paywalled full-text retrieval guarantees
