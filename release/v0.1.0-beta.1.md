# research-skills `v0.1.0-beta.1` Release Notes

Date: 2026-02-27
Stage: Beta

## Highlights

- Standardized triad orchestration (`codex` / `claude` / `gemini`) with automatic fallback.
- Added user-selectable runtime profile system (`profile_file`, stage-level profile overrides).
- Added `doctor` preflight mode for CLI/API-key/MCP command diagnostics.
- Added hard timeout + non-interactive runtime controls in base bridge.
- Added mock-based workflow tests for `parallel` and `task-run`.
- Added CI workflow to run compile + strict validator + unit tests.
