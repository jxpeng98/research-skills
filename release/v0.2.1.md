# research-skills `v0.2.1` Release Notes

Date: 2026-03-24
Stage: Stable

## Highlights (Draft)

- fix(installer): make `scripts/install_research_skill.sh` path normalization compatible with macOS `/bin/bash 3.2`, so `pipx`-driven upgrades no longer break when copying `.agent/workflows/*.md` and `templates/CLAUDE.project.md`
- test(installer): add a `/bin/bash` regression test that installs the Claude target into a temporary project path with spaces and verifies workflow and `CLAUDE.md` assets are copied correctly
- fix(release): align `scripts/pypi_preflight.sh` with the actual exported CLI aliases (`research-skills`, `rsk`, `rsw`) instead of probing a non-existent `rs` binary
- docs(release): update PyPI publishing guides in both docs and guides trees so release validation examples use `rsk`
- ci: add a Linux/Windows matrix job for validator and unit-test coverage while keeping shell and release-gate checks on Ubuntu
