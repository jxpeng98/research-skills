# research-skills `v0.3.0` Release Notes

Date: 2026-03-25
Stage: Stable

## Highlights (Draft)

- feat(skills): remove deprecated `version` from all `skills/*.md` frontmatter and simplify skill files to structural metadata plus execution guidance only
- feat(registry): make `skills/registry.yaml` the single canonical skill version source for release sync, validation, and documentation
- feat(validator): reject any skill frontmatter that reintroduces a `version` field, preventing future metadata drift
- docs(release): update release publishing guidance to reflect that version bumping now touches package metadata, portable workflow version, and registry only
